# Sistema de Leilões Online

## Descrição

O Sistema de Leilões Online é uma aplicação web desenvolvida em Python utilizando Flask.

O sistema permite que usuários criem leilões, realizem lances e acompanhem o encerramento automático dos leilões.

## Funcionalidades

* Cadastro e login de usuários
* Criação de leilões
* Edição e exclusão de leilões
* Upload de imagens
* Sistema de lances
* Histórico de lances
* Encerramento automático
* Definição automática do vencedor
* Painel administrativo
* Banimento de usuários
* Foto de perfil
* Categorias de leilões

## Tecnologias Utilizadas

* Python
* Flask
* Flask-Login
* Flask-SQLAlchemy
* Flask-Migrate
* SQLite
* HTML
* Bootstrap 5

## Como Executar

### 1. Clonar o repositório

git clone LINK_DO_REPOSITORIO

### 2. Entrar na pasta

cd sistemaDeLeilaoOnline

### 3. Criar ambiente virtual

python -m venv .venv

### 4. Ativar ambiente virtual

Windows:

.venv\Scripts\activate

### 5. Instalar dependências

pip install -r requirements.txt

### 6. Executar o projeto

python app.py
